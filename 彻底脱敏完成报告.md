## ⚠️ 重要：环境变量迁移

**您需要立即执行以下操作：**

### 步骤1：创建 `.env` 文件

在项目根目录创建 `.env` 文件，内容如下：

```ini
# 数据库配置
DB_PASSWORD=YOUR_DB_PASSWORD_HERE

# TikTok Token（请从之前的 config.py 迁移您的真实Token）
TIKTOK_ACCESS_TOKEN_1=YOUR_TIKTOK_TOKEN_1_HERE
TIKTOK_ACCESS_TOKEN_2=YOUR_TIKTOK_TOKEN_2_HERE

# TikTok App Secret
TIKTOK_APP_SECRET=YOUR_TIKTOK_APP_SECRET_HERE

# 代理配置（如果需要）
HTTP_PROXY=socks5h://127.0.0.1:10808
HTTPS_PROXY=socks5h://127.0.0.1:10808
```

### 步骤2：安装 python-dotenv（如果还没有）

```bash
pip install python-dotenv
```

### 步骤3：在代码中加载 .env

如果您的代码还没有加载 `.env` 文件，请在 `config.py` 开头添加：

```python
from dotenv import load_dotenv
load_dotenv()  # 加载 .env 文件
```
