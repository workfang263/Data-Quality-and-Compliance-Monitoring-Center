### 时区转换公式
```python
hours_to_add = 8.0 - source_timezone_offset
# 例如：UTC-8 → UTC+8：8.0 - (-8.0) = 16.0 小时
# 例如：UTC+8 → UTC+8：8.0 - 8.0 = 0.0 小时（不需要转换）
```
