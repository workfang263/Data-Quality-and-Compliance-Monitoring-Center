# 认证页视觉与交互回归清单

## 视图布局
- [ ] `>=1024px` 时登录页和注册页均为 50/50 分屏。
- [ ] `<1024px` 时左侧品牌区隐藏，右侧表单全宽显示。
- [ ] 登录页和注册页均不显示顶部业务导航栏。

## 表单状态
- [ ] 输入框默认/hover/focus 边框与阴影表现一致。
- [ ] 提交按钮具备正常态、hover、loading、disabled 四种状态。
- [ ] 错误提示使用 `el-alert` 且文案清晰可读。

## 鉴权流程
- [ ] 登录失败不会跳转，且页面显示后端错误信息。
- [ ] 登录成功后写入 `localStorage.token` 与 `localStorage.user`。
- [ ] 未登录访问受限页会跳转登录，登录后可按 `redirect` 回跳。
- [ ] 已登录访问 `/login` 或 `/register` 会跳转到 `/dashboard`。

## 主题可维护性
- [ ] 修改 `src/styles/auth.css` 中主色变量后，登录与注册按钮同步变化。
- [ ] 修改圆角变量后，输入框与按钮圆角同步变化。
