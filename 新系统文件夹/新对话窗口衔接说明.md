## 🎯 项目概述

**项目名称**：Streamlit → Vue + FastAPI 迁移项目

**目标**：将现有的 Streamlit 系统迁移到 Vue + FastAPI 架构，实现前后端分离。

**核心原则**：
1. ✅ **完全隔离**：新旧系统互不影响（方案二：复制代码）
2. ✅ **并行运行**：两个系统同时运行，不同端口
3. ✅ **共用数据库**：数据一致，无需迁移
4. ✅ **逐步切换**：功能模块逐步迁移，测试通过后再切换

**重要提醒**：
- ⚠️ **数据收集逻辑不变**：`data_sync.py`、`fb_spend_sync.py` 等脚本继续运行，不改动
- ⚠️ **功能完整保留**：所有功能都要有，只能更好不能缺少（如点击负责人名称弹出折线图）
- ⚠️ **UI简洁明了**：不要花里胡哨的文字和图案，保持简洁
