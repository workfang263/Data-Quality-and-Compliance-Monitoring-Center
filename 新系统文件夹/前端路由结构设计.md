# 前端路由结构设计

## 📋 说明

本文档定义了Vue前端应用的路由结构，包括：
- 路由路径
- 页面组件
- 路由守卫（权限控制）
- 路由元信息

**设计原则**：
1. 清晰的页面层级
2. 符合用户使用习惯
3. 支持路由守卫（登录验证）
4. 便于扩展和维护

---

## 🗺️ 路由结构

### 路由层级图
```
/ (根路径)
├── /login                    # 登录页面（公开）
├── /dashboard                # 看板页面（需要登录）
├── /owners                   # 负责人汇总页面（需要登录）
└── /stores                   # 店铺列表页面（需要登录）
```

## 📄 路由详细设计

### 1. 登录页面
**路径**：`/login`  
**组件**：`Login.vue`  
**权限**：公开（未登录用户可访问）  
**说明**：
- 登录成功后跳转到 `/dashboard`
- 如果已登录，访问此页面会自动跳转到 `/dashboard`

**路由配置**：
```typescript
{
  path: '/login',
  name: 'Login',
  component: () => import('@/views/Login.vue'),
  meta: {
    requiresAuth: false,  // 不需要登录
    title: '登录'
  }
}
```

## 🔒 路由守卫

### 全局前置守卫
**功能**：检查用户是否已登录

**逻辑**：
1. 如果路由需要登录（`meta.requiresAuth === true`）
2. 检查本地存储中是否有token
3. 如果没有token，跳转到 `/login`
4. 如果有token，验证token是否有效（可选：调用API验证）
5. 如果token无效，清除token并跳转到 `/login`

**实现位置**：`frontend/src/router/index.ts`

**代码示例**：
```typescript
router.beforeEach((to, from, next) => {
  // 检查路由是否需要登录
  if (to.meta.requiresAuth) {
    // 检查token
    const token = localStorage.getItem('token')
    if (!token) {
      // 没有token，跳转到登录页
      next({
        path: '/login',
        query: { redirect: to.fullPath }  // 保存原始路径，登录后跳转回来
      })
    } else {
      // 有token，允许访问
      next()
    }
  } else {
    // 不需要登录，直接访问
    next()
  }
})
```

---

### 登录后重定向
**功能**：登录成功后跳转到原始页面或默认页面

**逻辑**：
1. 登录成功后，检查URL参数中是否有 `redirect`
2. 如果有 `redirect`，跳转到该路径
3. 如果没有，跳转到 `/dashboard`

**代码示例**：
```typescript
// 在Login.vue中
const handleLogin = async () => {
  // ... 登录逻辑
  const redirect = route.query.redirect as string || '/dashboard'
  router.push(redirect)
}
```

## 📋 路由配置完整示例

```typescript
// frontend/src/router/index.ts
import { createRouter, createWebHistory } from 'vue-router'
import type { RouteRecordRaw } from 'vue-router'

const routes: RouteRecordRaw[] = [
  {
    path: '/',
    redirect: '/dashboard'
  },
  {
    path: '/login',
    name: 'Login',
    component: () => import('@/views/Login.vue'),
    meta: {
      requiresAuth: false,
      title: '登录'
    }
  },
  {
    path: '/dashboard',
    name: 'Dashboard',
    component: () => import('@/views/Dashboard.vue'),
    meta: {
      requiresAuth: true,
      title: '数据看板'
    }
  },
  {
    path: '/owners',
    name: 'Owners',
    component: () => import('@/views/Owners.vue'),
    meta: {
      requiresAuth: true,
      title: '负责人汇总'
    }
  },
  {
    path: '/stores',
    name: 'Stores',
    component: () => import('@/views/Stores.vue'),
    meta: {
      requiresAuth: true,
      title: '店铺列表'
    }
  },
  {
    path: '/:pathMatch(.*)*',
    name: 'NotFound',
    component: () => import('@/views/NotFound.vue'),
    meta: {
      requiresAuth: false,
      title: '页面未找到'
    }
  }
]

const router = createRouter({
  history: createWebHistory(),
  routes
})

// 路由守卫
router.beforeEach((to, from, next) => {
  // 设置页面标题
  document.title = (to.meta.title as string) || 'Shoplazza Dashboard'
  
  // 检查是否需要登录
  if (to.meta.requiresAuth) {
    const token = localStorage.getItem('token')
    if (!token) {
      next({
        path: '/login',
        query: { redirect: to.fullPath }
      })
    } else {
      next()
    }
  } else {
    next()
  }
})

export default router
```
